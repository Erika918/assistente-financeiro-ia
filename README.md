# assistente-financeiro-ia

Projeto desenvolvido em Python com Flask para criar uma experiência digital financeira guiada por IA generativa.

## Objetivo

Oferecer um assistente financeiro simples, claro e interativo, capaz de responder dúvidas, explicar conceitos financeiros e realizar simulações básicas.

## Tecnologias utilizadas

- Python
- Flask
- HTML
- CSS
- JavaScript
- IA Generativa
- UX/UI

## Funcionalidades

- Chat financeiro inteligente
- FAQ sobre Pix, cartão, juros e organização financeira
- Simulador de parcelas
- Interface moderna e responsiva

## Como executar

```bash
pip install flask
python app.py
